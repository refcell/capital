# Projections

Concise Value Line style projection sheets generated from the workbook models in `models/`.

Default output files should be named `TICKER.md` and should show separate bear, base, and bull tables with 1-, 2-, 3-, 5-, and 10-year projections for revenue, revenue growth, operating income, net income, FCF yield, stock price, P/E, and stock-price CAGR from the current price. Stock prices should work backward from an explicit year-10 terminal P/E anchor for each case. Each file should note the model workbook used, the quote date, and any extrapolation beyond the explicit forecast period.
