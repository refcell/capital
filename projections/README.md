# Projections

Concise Value Line style projection sheets generated from the workbook models in `models/`.

Default output files should be named `TICKER.md` and should show 1-, 2-, 3-, 5-, and 10-year projections for revenue, operating income, net income, FCF yield, stock price, and P/E. Each file should note the model workbook used, the quote date, and any extrapolation beyond the explicit forecast period.
