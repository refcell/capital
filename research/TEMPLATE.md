## How To Use This Template

This template is meant to standardize the questions, not flatten the writing. Each report should have a clear story, company-specific sections, and source-backed numbers.

Use these rules:

- Start each major section with a narrative paragraph before tables.
- Make the core investment debate obvious in the first page.
- Separate what the market believes from what the report believes.
- Use the right valuation framework for the business model.
- Put analyst expectations, management guidance, and historical evidence behind forward projections.
- Explain why each bear, base, and bull projection is plausible.
- Do not hide important uncertainty in a table. If a number is fragile, say why.
- Note source dates, access limits, and conflicting data-provider figures.
- Prefer primary sources for filings, guidance, share count, debt, segment data, risks, and management commentary.
- Use public analyst-consensus sources only as expectations anchors, not as proof of intrinsic value.

Delete this "How To Use" section in completed reports.
